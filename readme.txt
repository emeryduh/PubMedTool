Installation
------------
1. Install Nodejs at https://nodejs.org/en/
2. Open your CLI and navigate to the project directory
3. type "npm install" (without the quotes)
4. type "node index.js"

------------

If you have used Node before then update your Node environment by running the following commands
npm install -g node
npm install -g npm
npm install -g node-gyp

------------

If you do not have C/C++ and Python build tools installed then go to https://github.com/nodejs/node-gyp#installation
to install.  The link will have instructions based on your OS.
